# SpecStudio Roadmap

## Prototype

- Static app shell
- Single-user project creation
- Manual intake form
- Mocked AI output
- Markdown export

## MVP

- Authentication
- Workspace roles
- Real run lifecycle
- AI provider integration
- Review and approval flow
- Audit events
- JSON and Markdown export

## Production v1

- Integration handoff
- Admin settings
- Observability dashboard
- Retention controls
- End-to-end test suite
- Deployment and rollback runbooks

## Later

- Team templates
- Advanced analytics
- Marketplace or package sharing
- Enterprise SSO
- Custom workflow builder
